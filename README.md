# 💰 Expense Tracker


**Expense Tracker** is a colorful, interactive web app to track income 💵 and expenses 💸 in real-time.
Visualize your spending habits, manage your budget, and stay on top of your finances effortlessly!

## ✨ Features

- 📝 Add Transactions: Record income or expenses with description and amount

- ❌ Delete Transactions: Remove unwanted entries easily

- 💹 Real-time Balance: Updates total balance, income, and expenses instantly

- 💾 Persistent Data: Saves all transactions using localStorage

- 📱 Responsive Design: Works perfectly on desktop, tablet, and mobile

------

## 🛠️ Technologies Used

- HTML5 – Clean semantic markup

- CSS3 – Modern and colorful styling

- JavaScript – Dynamic behavior and localStorage
------

## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/Sadaqath18/Expense-Tracker.git 

```
2. Navigate to the project folder
```bash
cd Expense-Tracker
```

- Open the app in your browser

- Open index.html with any modern browser (Chrome, Firefox, Edge, Safari).
